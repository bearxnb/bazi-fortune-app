# AI 易经八字年度运势（网页版 MVP）

## 运行方式

1. 最简单：直接用浏览器打开 `index.html` 即可使用。
2. 推荐（便于 API 调用与复制权限）：在项目目录执行 `python3 -m http.server 8080`，然后访问 `http://localhost:8080/bazi-fortune-app/`。

## 当前功能

- 输入姓名、出生年月日时、国家、城市。
- 自动带出时区与经纬度，并按输入日期动态计算时区偏移（兼容夏令时）。
- 基于台湾常见子平法口径（立春分年、节气分月、真太阳时）计算四柱八字（年/月/日/时）。
- 统计命盘五行比例，并生成命局五行分析与性格分析。
- 分析姓名各字五行与命盘联动关系。
- 生成年度运势报告（2026 大势 + 个人承接逻辑 + 事业/爱情/健康/财富）。
- 输出注意事项与转运建议。
- 一键复制纯文本报告（便于粘贴到微信、邮件、文档）。
- 可选接入 OpenAI API Key，生成 AI 增强解读。
- 内置后台验证：立春换年、节气定月、日柱连续性、时柱边界、六十甲子唯一性。
- 中国城市扩展为 50 个经济发达城市，并按拼音首字母排序。

## 说明

- 本版本为可运行 MVP，适合需求验证和演示。
- 城市数据库为内置示例，可扩展为完整全球地理库或第三方地理编码服务（含更精细行政区）。
- 节气当前为工程近似到“日”的实现；如需专业级排盘一致性，可升级为天文历算到“分”。
- 姓名五行为工程分析法（常见字映射 + 编码兜底），可后续升级为康熙字典笔画/字源系统。
- OpenAI Key 在前端本地直连模式下仅保存在当前页面内存中，不会写入后端数据库。
